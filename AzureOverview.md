## Azure Overview
1. 
  - Infrastructure-as-a-Service VM
  - Platform-as-a-Service Azure APP Service

2. Compute:
  - Virtual Machines
  - App Service
  - Container Instances
  - Kubernetes Services
  - Functions

3. Storage:
  - Blob Storage: Unstructured
    - Hot, Cool, Archive
  - File Storage: Hierarchical, can mount on Windows servers
  - Data Lake: Hierarchical, Hadoop-compatible storage, data analytics applications
  - SQL Database: Relational Databases
  - MySQL/MariaDB/PostgreSQL: Open Source
  - Synapse Analytics: Data Warehouse
  - Cosmos DB : No SQL
  - Cache for Redis: No SQL

4. Networking:
  - VNet
  - Subnets
  - VNet peering
  - VPN
  - Express Route: Dedicated connection

5. 
  - Billing Account/Subscription/Resource
  - Billing Accounts can have multiple subscriptions
  - Resource Groups can have multiple Resources

6. 
  - Available Region: contains 3 availability zone
  - Availability Zone

7. 
  - Migration:
    - Azure Migrate: Hyper-V/VMware/
    - Azure Active Directory
  - DevOps
    - Pipelines
    - DevTest Labs: Admin-controlled costs
    - CDN
  - IoT
    - IoT Central
    - IoT Hub
    - Sphere: Security
  - Analytics
    - HD Insight: Hadoop/Spark/Hive/Strom
    - Databricks: User-friendly Spark
    - Synapse Analytics: Data warehouse with support for Spark
  - AI
    - Cognitive Services
    - Bot Service
    - Machine Learning Studio
    - Machine Learning Services
  - Integration
    - Logic Apps: detect event
    - Event Grid: notify Logic Apps
